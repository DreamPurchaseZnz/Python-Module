: colon
======
## One and Two

```
array = [1, 2, 5, 3, 6, 8, 4]
>>array[0:]  # 列出第一个数字之后的所有               >>array[0::] # 与array[0:]结果相同
>>[1, 2, 5, 3, 6, 8, 4]                             >>[1, 2, 5, 3, 6, 8, 4]

>>array[:-1] # 倒数最后一个数字之前的所有             >>array[::-1] # 从最后一位每隔负一步取一次等价于Reverse[1, 2, 5, 3, 6, 8]
>>[1, 2, 5, 3, 6, 8]                                >>[4, 8, 6, 3, 5, 2, 1]

>>array[:2] # 从第一个到第二位[0,2)                  >>array[::2]  # 从第一位每隔两步取一次
>>[1, 2]                                            >>[1, 5, 6, 4]
```
