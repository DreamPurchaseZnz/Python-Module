# Dictionaries in python
## Basic Operation
### Store the variables
```
released = {
		"iphone" : 2007,
		"iphone 3G" : 2008,
		"iphone 3GS" : 2009,
		"iphone 4" : 2010,
		"iphone 4S" : 2011,
		"iphone 5" : 2012
	}
```

### Add the variables
```

released["iphone 5S"] = 2013
```
### Remove a key and it's value
```
del released["iphone"]
```

## Check the length
use the "len()"

### Get variables

use the key to search the value
```
a = released["iphone 5"]
```
